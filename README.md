# Bioinformatics Ontology Knowledge Graph (KG)

## 📌 Project Overview

This project constructs a **Knowledge Graph (KG)** from bioinformatics ontology data. It integrates gene-ontology relationships and performs network analysis to extract meaningful biological insights.

---

## 🎯 Objectives

* Parse biological ontology data
* Map gene-to-ontology relationships
* Construct a knowledge graph
* Analyze graph structure (centrality, degree distribution)
* Visualize biological networks

---

## 🧠 Key Features

* Ontology parsing (GO data)
* Gene-to-GO mapping
* Graph construction using Python
* Centrality analysis
* Network visualization

---

## 📂 Project Structure

* `scripts/` → Core Python scripts
* `processed/` → Cleaned datasets
* `outputs/` → Graphs, metrics, and results
* `logs/` → Execution logs

---

## ⚙️ How to Run

```bash
python scripts/run_all.py
```

---

## 📊 Outputs

* Knowledge Graph (.graphml)
* Node & Edge datasets
* Centrality metrics
* Visualization plots

---
